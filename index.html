You will imagine a XXXX Number in his head and the pc will try to discover it:
  • The Number must not start by the digit 0
  • The Number is composed of 4 digits (from 0 to 9)
  • The Number must contain all different digits (no repetition, exp. 0353 is prohibited)
  • If the pc proposes a number that contains: 0 digit of the Number XXXX,you must say : 0 bulls 0 cows
  • 1 digit that belongs to the number XXXX, and this number is not in the right position, you must say 1 cow and 0 bulls (Exp: XXXX= 1245,pc proposes 7856)
  • 2 digits that belong to the number XXXX, and these 2 digits are not in the right position,you must say 2 cows and 0 bulls (Exp : XXXX= 1245, someone proposes 4589) etc.
  • A number that belongs to the number XXXX, and this number is in the right position,you must say 1 bull and 0 cows (Exp: XXXX= 1245, someone proposes 1763) etc.
  • The pc has only seven tries to find the number XXXX
